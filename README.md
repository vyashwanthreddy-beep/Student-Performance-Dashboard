# Student Performance Dashboard

## Overview

The Student Performance Dashboard is a Power BI project designed to analyze and visualize student academic performance across various subjects, demographics, and behavioral factors.

This dashboard helps educators and analysts identify trends, patterns, and key factors affecting student outcomes.

---

##  Objectives

* Track student performance across subjects
* Identify high-performing and low-performing students
* Analyze impact of factors like attendance, study time, and parental education
* Enable data-driven decision making in education

---

##  Files Included

* `Student Performance Dashboard.pbix` – Power BI dashboard file
* `table.csv` – Dataset used for analysis
* `README.md` – Project documentation

---

## Key Metrics

* Average Marks
* Pass/Fail Rate
* Subject-wise Performance
* Attendance Impact
* Study Hours vs Scores

---

##  Features

* Interactive filters (gender, class, subject)
* Visual charts (bar, line, pie)
* KPI indicators
* Drill-down analysis

---

## 🧾 Dataset Description

The dataset contains student-level data including:

* Demographics (Gender, Age)
* Academic scores (Math, Science, English)
* Behavioral metrics (Study Hours, Attendance)

---

##  Tools Used

* Power BI
* Microsoft Excel / CSV

---

##  How to Use

1. Open the `.pbix` file in Power BI Desktop
2. Load the dataset if not already connected
3. Interact with filters and visuals
4. Analyze insights from dashboards

##  Future Improvements
